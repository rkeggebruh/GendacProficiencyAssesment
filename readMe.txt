# Product Management Application

## Setup Instructions

1. Ensure you have a web server to host the files (e.g., using VS Code Live Server extension or any HTTP server).
2. Place all the files in the project directory.
3. Open `index.html` in your web browser.

## Dependencies

- jQuery (included via CDN in `index.html`)

## Functionality

- Displays a list of all products
- Allows creation of a new product
- Enables updating an existing product
- Facilitates deletion of a product
- Shows messages for actions performed (creation, update, deletion)
